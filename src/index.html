How to use Configuration WEBPACK-4

1.Download Configuration WEBPACK-4 from GitHub;

2.Install Node Modules: npm i;

3.Run the template:
      npm run start (server startup)
      npm run dev (create a dist directory and expand the project file structure)
      npm run build (structure for production)
